# ui-design

Sanbao UI design skills and a sample high-fidelity fitness check-in app mockup.

## Included skills

- `skills/sanbao-product-manager`: clarifies product requirements before UI design.
- `skills/sanbao-ui-design`: generates confirmation-gated UI design boards and HTML handoff mockups.

## Sample mockup

- `index.html`: static high-fidelity iOS-style fitness check-in mockup.
- `styles.css`: visual styling for the mockup.
- `assets/fitness-checkin-design-board.png`: approved design reference.
- `screenshots/fitness-checkin-html.png`: rendered HTML verification screenshot.
